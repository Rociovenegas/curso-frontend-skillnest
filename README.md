# Desarrollo Front-End con JavaScript

## Sobre el curso

El curso cubre los fundamentos del desarrollo web (Front-End), desde ¿Qué es la web? hasta JavaScript, viendo en el camino HTML, CSS y algoritmos. El curso se imparte en línea con clases en vivo y con tareas prácticas.

## Repositorio

Este repositorio es donde el estudiante (yo xd) ha almacenado las tareas y actividades. Las actividades están divididas en tres módulos:

* **HTML** - Estructura y contenido
* **CSS** - Estilos y diseño
* **JavaScript** - Interactividad y lógica

Cada actividad es un proyecto independiente.
